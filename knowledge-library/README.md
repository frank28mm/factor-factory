# Factor Factory Knowledge Bootstrap

This directory contains public bootstrap assets and sync tools.

It intentionally does not include any previous user's account state, alpha ids,
simulation results, submission history, cookies, or browser profiles.

## Included

- Public starter field metadata generated from `config/fields.yaml`.
- WorldQuant official metadata sync entrypoints for users who have their own
  logged-in WorldQuant BRAIN session.

## Not Included

- Historical Factor Factory `state/`.
- User-specific WorldQuant result ledgers.
- Submitted or unsubmitted alpha records from another account.
